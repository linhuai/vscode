# vscode 


## 常用快捷键

### 多项选择

1. **⌥⌘↓** 或 **⌥⌘↑** 将光标插入到下方或上方。

2. **⌘D** 选择光标处的单词，或当前选择的下一个出现位置
3. **⇧⌘L** 设置选择的大小写
4. **⇧⌥** + click 添加光标

### 收缩/扩展选择

1. **^⇧⌘←** 和 **^⇧⌘→** 快速缩小或扩展当前选择

2. ⇧⌥⌘↓	列选择向下	
3. ⇧⌥⌘↑	列选择向上	
4. ⇧⌥⌘←	列选择左
5. ⇧⌥⌘→	列选择右
6. ⇧⌥⌘PageDown	列选择向下翻页	
7. cursorColumnSelectPageDown
8. ⇧⌥⌘PageUp	列选择向上翻页
